Best Sellers Book API

Overview

The purpose of this project is to display books that have been in the New York Times's bestseller lists in the past. It uses two API sources: 1) The New York Times Book API and the Open Library API. The New York Times API is used to retrieve a list of books based on 3 filters. User can query books based on author, age group, and publisher. Users have the option to query using these filters individually, together, or none at all. Once the books data is retrieved, the isbn10 numbers of each book will be used to get their respective images from Open Library API. The results will then be displayed in an HTML table. For the sake of brevity, only 10 records will be displayed, even if the query has a lot of results.
