What is this???

These are all web components!!!! They are custom made HTML5 markup that can be created and re-used on the web. The source code for these can be written once then imported into a project. This greatly improves productivity for developers if you encounter the same use cases over and over.

Use Case

There are many web design patterns that use the same ideas for certain things. Card design is a popular one use to list people. One example would be an organization chart of a company. This example here is the org chart from Walmart. With web components, you can write your code once and re-use them as many times as needed. #productivity
